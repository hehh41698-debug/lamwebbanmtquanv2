## 📋 Giới Thiệu

Computer Store là một website thương mại điện tử bán máy tính và linh kiện điện tử được xây dựng bằng **Vue 3** và **Express.js**. Dự án cung cấp đầy đủ các chức năng của một cửa hàng trực tuyến với phân quyền người dùng và quản trị viên.

### ✨ Tính Năng Nổi Bật

#### 👤 Khách hàng (User)
- ✅ Đăng ký / Đăng nhập tài khoản
- ✅ Xem danh sách sản phẩm với bộ lọc và tìm kiếm
- ✅ Xem chi tiết sản phẩm với hình ảnh và thông số kỹ thuật
- ✅ Thêm sản phẩm vào giỏ hàng
- ✅ Quản lý giỏ hàng (thêm, xóa, cập nhật số lượng)
- ✅ Đặt hàng với nhiều phương thức thanh toán
- ✅ Xem lịch sử đơn hàng và trạng thái
- ✅ Đánh giá sản phẩm
- ✅ Quản lý thông tin cá nhân

#### 🔐 Quản trị viên (Admin)
- ✅ Dashboard thống kê doanh thu và đơn hàng
- ✅ Quản lý sản phẩm (CRUD)
- ✅ Quản lý danh mục sản phẩm
- ✅ Quản lý đơn hàng và cập nhật trạng thái
- ✅ Quản lý người dùng (khóa/mở khóa)
- ✅ Quản lý đánh giá sản phẩm
- ✅ Xem báo cáo và thống kê

## 🛠️ Công Nghệ Sử Dụng

### Frontend
- **Vue 3** - Framework JavaScript progressive
- **Vue Router** - Quản lý routing
- **Pinia** - State management
- **Bootstrap 5** - UI Framework
- **Axios** - HTTP client
- **Vuelidate** - Form validation
- **Vue3 Toastify** - Toast notifications
- **Vue3 Carousel** - Carousel slider

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **MongoDB** - Database
- **Mongoose** - ODM for MongoDB
- **JWT** - Authentication
- **Bcrypt.js** - Password hashing
- **Multer** - File upload
- **Nodemailer** - Email service
- **Cloudinary** - Image storage
- **Passport.js** - Authentication strategies
